# 视觉反馈与打击感方案（Juice Design）

> **角色**：Whimsy Injector + Creative Director
> **版本**：v1.0 | 日期：2026-03-11
> **原则**：每一帧都在讲述一个微型故事

---

## 设计哲学

> "SNKRX 的方块不是方块，它们是有个性的角色。  
> 打击感不来自画面复杂度，来自**精确的时间节奏**。"

极简视觉的"爽感"公式：
```
爽感 = 打击停顿(帧精度) + 粒子密度(爆发感) + 音效节奏(同步) + 色彩对比(冲击感)
```

## 1. 打击停顿（Hit Stop / Hitstun）

### 斩击（近战）

```
触发时机：攻击命中的第一帧

时长：3–5 帧（@60fps = 50–83ms）
  └─ 普通攻击：3帧（快速，节奏感强）
  └─ 重击/暴击：6帧（明显停顿，感受"重量"）
  └─ 白金羁绊大招：10帧（戏剧性停顿）

实现方式：
  hit_stop_timer = 5
  on_update():
    if hit_stop_timer > 0:
      hit_stop_timer -= 1
      skip_physics_update()  // 仅物理暂停，特效继续播放
      return

视觉强化：
  ├─ 攻击方：在停顿帧做轻微"前冲"位移（+4px 向目标方向）
  ├─ 受击方：小幅"压扁"变形（scale_x *= 1.3, scale_y *= 0.7）
  └─ 屏闪：白色半透明全屏叠加层，透明度从 40% 线性降至 0%（持续 2帧）
```

### 远程攻击（子弹命中）

```
触发时机：子弹消失的同一帧

时长：2帧（比近战短，体现"速度感"而非"重量感"）

视觉强化：
  ├─ 子弹命中点：爆发光圈扩散（半径 0 → 32px，持续 4帧，颜色=子弹颜色）
  ├─ 无全屏屏闪（远程攻击频率高，避免视觉疲劳）
  └─ 受击方：小幅"后飞"位移（沿子弹方向 +8px，持续 3帧后弹回）

特殊情况：弹射子弹（法师金羁绊）
  └─ 每次弹射：新的衍射光圈（颜色渐变：蓝→紫→红，表示弹射次数）
```

## 2. 死亡特效：方块碎裂粒子系统

### 核心逻辑

```
on_enemy_death(enemy):

  // Step 1: 主体碎裂
  spawn_shards(
    count: 8–12,
    shape: "方形碎片"（enemy颜色 + 随机明度±20%）,
    size: enemy.size × random(0.2, 0.5),
    velocity: 随机方向，速度 80–200 px/s,
    rotation_speed: random(-360, 360) deg/s,
    lifetime: 0.4–0.8s,
    gravity: 150 px/s²（向下拉扯）
  )

  // Step 2: 能量爆散（"灵魂"效果）
  spawn_energy_particles(
    count: 6–8,
    shape: "小圆点"（白色/亮色）,
    velocity: 向外扩散，速度 60–120 px/s,
    lifetime: 0.3s,
    ease: "easeOut"（快速扩散后消失）
  )

  // Step 3: 地面残影
  spawn_shadow(
    shape: enemy的轮廓，颜色=enemy颜色×50%透明,
    lifetime: 0.5s,
    fade: 线性淡出
  )
```

### 不同死亡情境的差异化特效

| 死亡原因 | 特效变化 |
|---|---|
| 普通击杀 | 标准碎裂（8片，中速） |
| 暴击击杀 | 碎片数量×2，速度×1.5，屏闪 1帧 |
| 爆炸击杀 | AOE 范围内所有死亡：同步碎裂 + 中央橙色冲击波圈 |
| 毒/燃烧击杀 | 碎片带对应颜色（绿/橙），碎片消失时留下小火花/毒液点 |
| 弹射击杀 | 碎片带弹射轨迹的拖尾颜色 |

## 3. 羁绊激活视觉提示

### 触发条件
玩家蛇上的某职业单位数量达到羁绊阈值（2/4/6人）的**瞬间**。

### 三级激活演出

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🥈 银级羁绊激活（2人）— 低调确认感
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  时长：0.5s
  参与单位：相关职业单位轻微发光（颜色=职业主题色，亮度提升40%）
  UI：右上角图标出现，职业 icon 从灰色变为彩色（0.3s 弹入动画）
  音效：轻快的"叮"声（短促，不打断游戏节奏）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🥇 金级羁绊激活（4人）— 中等成就感
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  时长：1.0s
  参与单位：
    ├─ 相关单位：脉冲发光（向外扩散光环 × 2次，半径 0→64px）
    └─ 光环颜色：职业主题色（战士=红，法师=蓝，等）
  
  全局效果：
    ├─ 屏幕边缘短暂暗化（vignette 效果，0.3s 后恢复）
    └─ 蛇的轮廓出现同色轮廓线（持续 1s 后渐隐）
  
  UI：
    ├─ 羁绊 icon 升级动画（扩大120%后弹回，金色光芒）
    └─ 屏幕顶部短暂显示文字："[职业名] 羁绊·金 已激活" （淡入淡出，1.5s）
  
  音效：中等质感的"铮"声 + 低频共鸣

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
💎 白金羁绊激活（6人）— 史诗级成就感
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  时长：2.5s（游戏时间暂停 0.5s，给玩家欣赏时间）

  阶段分解：
  
  [0s – 0.5s] 凝结时刻
    └─ 游戏慢动作（time_scale = 0.1）
    └─ 整条蛇：从蛇尾到蛇首依次点亮（波浪式光效，延迟 0.1s/格）
  
  [0.5s – 1.5s] 爆发时刻
    ├─ 所有相关单位同时爆发大光环（半径 0→200px，持续 1s）
    ├─ 屏幕全局色调闪烁（职业主题色，透明度 60%→0%，0.3s）
    ├─ 粒子从蛇身向外喷射（职业主题色碎星，50个，扩散后消失）
    └─ 相机轻微震动（振幅 6px，频率 20hz，持续 0.5s）
  
  [1.5s – 2.5s] 余韵时刻
    ├─ 蛇保持持续光晕（职业色，透明度 30%，直到羁绊被打破）
    ├─ UI：职业 icon 变为白金色，周围持续旋转小星星（idle 动画）
    └─ 屏幕顶部显示："⬡ [职业名] 精英 · 已全员就位" （较大字体，2s）
  
  音效：
    ├─ 慢动作阶段：低频共鸣（心跳感）
    ├─ 爆发瞬间：冲击音效 + 和声"啊——"（短促）
    └─ 余韵：轻柔的持续颤音（告知玩家增益持续存在）
```

## 4. 其他细节 Juice 清单

| 元素 | 效果 |
|---|---|
| 金币拾取 | 金币向 UI 金币栏飞去（抛物线轨迹，带拖尾） |
| 蛇身增加新成员 | 新成员从蛇尾位置"弹出"（scale 0→1.3→1，0.3s） |
| 蛇首移速 | 转向时轻微"甩尾"（后几格延迟更大，强调弹性感） |
| 连击数 | 屏幕角落连击计数器，字体随连击数增大（最大1.5×） |
| 关卡结束 | 剩余敌人全部爆炸（以蛇首为圆心，由近到远依次死亡） |

